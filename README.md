# Voyagent 
Welcome to Voyagent Back-end. This project serves as the backend for Voyagent Website

# Technologies Used
1. JavaScript
2. Express.js
3. Node.js
4. Visual Studio Code

# Project Root

```
- css

- js
  - file.js
```

# Usage
To install npm before running back-end server :
```
npm install
```
To run the back-end server :
```
npm start
```
